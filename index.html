Micro:bit ↔ Vending Machine Test Console

page to test the serial messaging between the browser and a micro:bit (if you have the Hummingbird). Open the console, connect over Web Serial, and send text commands like LED ON or SERVO 1 90. The MicroPython test can use a few basic commands. Press button A on the micro:bit to have the page pick 0 or 1 and send the result back as CHOICE 0 / CHOICE 1; the micro:bit should show the number on its LEDs.

Requires a Chromium-based browser with the navigator.serial API (it don't work with others sadly). The micro:bit should appear as a USB serial device (vendor ID 0x0D28). When connected, this console sends newline-terminated text commands and shows responses from the board.

Send a command

Commands are sent with a trailing newline. The test I madew supports: LED ON, LED OFF, SERVO 1 0-180, and SENSOR? (returns light reading).
