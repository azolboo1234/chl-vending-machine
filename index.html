Candy Vending Machine

Press button A on the micro:bit to spin a one-column candy vending machine. The browser weights the spin toward the chosen preference (75% preferred candy, 15% the other option, 10% no candy), shows the pulled item, and sends the result back to the micro:bit so the Hummingbird can react.

Bypass: off

Use a Chromium-based browser with navigator.serial. Only the host computer with the micro:bit plugged in should connect; guests can still watch the log via the join code.
